Aplicación de Videollamadas

¡Conéctate con tus amigos y familiares en cualquier momento y en cualquier lugar!

Funcionalidades

  • Realizar videollamadas con tus contactos.
  • Grabar llamadas para revivir momentos especiales.
  • Escuchar audios grabados en cualquier momento.
  • Añadir y gestionar contactos para mantenerse en contacto fácilmente.
  • Unirse a salas públicas de tus contactos para socializar.
  • Crear tus propias salas para videollamadas grupales.
  • Ver el estado de conexión de tus contactos para saber quién está disponible.
  • Guardar audios temporales en la caché del navegador o permanentemente en la base de datos.

Tecnologías Utilizadas

Frontend

  • Angular 17
  • Angular Material
  • Tailwind CSS
  • WebRTC
  • APIs del navegador: Cache Storage, Local Storage, MediaRecorder, MediaDevices
  • Manejo de streams de video y audio

Backend

  • Spring Boot
  • Spring Data
  • Spring Security
  • AWS S3
  • JWT
